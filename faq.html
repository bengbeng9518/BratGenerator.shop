Frequently Asked Questions

Find answers to common questions about using Brat Generator

Getting Started

How do I create my first text effect?

Creating your first text effect is simple:

  1. Enter your text in the input field
  2. Choose a color from the color picker
  3. Preview your text effect in real-time
  4. Click the download button to save your creation
What file formats are supported for download?

Currently, all text effects are downloaded as high-quality PNG files with transparent backgrounds. This format ensures maximum compatibility and quality for both web and print use.

Features & Customization

Can I use custom colors?

Yes! While we provide a curated selection of colors, you can use any custom color by selecting from our color picker. This gives you unlimited possibilities for your text effects.

Troubleshooting

Why can't I download my text effect?

If you're having trouble downloading:

  • Check your internet connection
  • Make sure you've entered some text
  • Try using a different browser
  • Clear your browser's cache

If the problem persists, please contact our support team.

The colors aren't displaying correctly

Color display issues can be caused by:

  • Browser color profile settings
  • Monitor calibration
  • Operating system color settings

Try updating your browser or adjusting your display settings.

Usage Rights

Can I use the text effects commercially?

Yes! All text effects created with Brat Generator are free to use for both personal and commercial purposes. No attribution is required, though it's always appreciated!

Is there a limit to how many effects I can create?

No, there are no limits! You can create as many text effects as you want, completely free of charge.
